# Gabaritos oficiais

Use um arquivo por prova, com o nome `gabarito_NUMERO.txt`.

Exemplo: `gabarito_46.txt`

Formato recomendado:

```txt
1 A
2 C
3 D
4 B
5 A
```

Tambem sao aceitos `1. A`, `1 - A`, `1:A` e `Questao 1: A`.

Regras:

- use somente alternativas `A`, `B`, `C` ou `D`;
- mantenha uma linha por questao;
- nao pule numeros;
- confira se a numeracao bate com a prova importada;
- se houver anulacao, escreva `ANULADA` no lugar da letra.
